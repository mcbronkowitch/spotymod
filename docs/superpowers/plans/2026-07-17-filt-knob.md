# FILT — bipolarer Cutoff-Trim: Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ein bipolarer FILT-Knopf pro Part (−1…+1, Mitte neutral), der mit der FILTER-Lane summiert: links Cutoff zudrehen bis in Stille, rechts öffnen — ohne neuen Filter, null Per-Sample-Kosten.

**Architecture:** `SynthEngine` addiert den Reglerwert als Offset im normalisierten Lane-Raum vor `filter_hz()` (linke Hälfte skaliert ×1.25); läuft der Offset unter die 60-Hz-Schiene, blendet ein control-rate berechneter `_filt_gain` den Part über den vorhandenen LEVEL-OnePole (10 ms) klickfrei in Stille. Push-Kette exakt wie RES (`Instrument` → `Part` → `SynthEngine`). VCV bekommt den sechsten Voice-Row-Knopf; die Param-IDs `FILT_A/_B` werden ans Ende der PARAMS-Liste angehängt (Patch-Kompatibilität, wie CHOKE).

**Tech Stack:** C++17 portable engine (`engine/`), doctest (`tests/`, clang+ninja via `env.sh`), VCV Rack SDK 2.6.6 host (`host/vcv/`, MSYS2 make + WinLibs gcc), panel generated by `host/vcv/res/gen_panel.py`.

**Spec:** `docs/superpowers/specs/2026-07-17-filt-knob-design.md`

static void feed(SynthEngine& e, float pitch, float filter = 1.f, float level = 1.f) {
    float t[LANE_COUNT] = { 0.f, filter, pitch, 0.f, level };
    e.set_targets(t, 0.5f);

// Fresh engine in "measurement" trim: pure sine, no sub, no detune.
static void fresh(SynthEngine& e, uint32_t seed = 99) {
    e.set_seed(seed);
    e.init(48000.f);
    e.set_sub(0.f);
    e.set_detune(0.f);
    e.set_cycle(1.f);
    feed(e, 0.5f);
}

static std::vector<float> render_l(SynthEngine& e, int n) {
    std::vector<float> out(n);
    for (auto& s : out) {
        float l = 0.f, r = 0.f;
        e.process(l, r);
        s = l;
    }
    return out;
}

static float rms(const std::vector<float>& v, size_t from) {
    double acc = 0.0;
    for (size_t i = from; i < v.size(); ++i) acc += (double)v[i] * v[i];
    return std::sqrt((float)(acc / (double)(v.size() - from)));
}

TEST_CASE("filt: 0 is bit-identical on the engine") {
    SynthEngine a, b;
    fresh(a);
    fresh(b);
    b.set_filt(0.f);
    a.trigger(0.5f);
    b.trigger(0.5f);
    auto va = render_l(a, 48000);
    auto vb = render_l(b, 48000);
    for (int i = 0; i < 48000; ++i) REQUIRE(va[i] == vb[i]);
}

TEST_CASE("filt: full left is silent for any lane position") {

        fresh(e);
        feed(e, 0.5f, lane);
        e.set_filt(-1.f);
        e.trigger(0.5f);

        // 10-ms-OnePole: nach 0,2 s ist der Rest < e^-20; -80 dBFS ist grosszuegig
        CHECK(rms(v, 9600) < 1e-4f);
    }
}

TEST_CASE("filt: sweep through the whole range is click-free") {
    SynthEngine e;
    fresh(e);
    feed(e, 0.f, 0.5f);               // 110 Hz: kleines natuerliches Sample-Delta
    e.trigger(0.f);
    render_l(e, 4800);                // Attack ausklingen lassen
    float maxstep = 0.f, prev = 0.f;
    const int blocks = 1000;          // 2 s Sweep in Control-Block-Schritten
    for (int bi = 0; bi <= blocks; ++bi) {
        e.set_filt(-1.f + 2.f * bi / blocks);
        for (int i = 0; i < SynthEngine::kCtrlInterval; ++i) {
            float l = 0.f, r = 0.f;
            e.process(l, r);
            if (bi > 0 || i > 0) maxstep = std::max(maxstep, std::abs(l - prev));
            prev = l;
        }
    }
    CHECK(maxstep < 0.05f);           // Klicks waeren Spruenge >> Signal-Delta
}

static float inst_rms(Instrument& inst, int samples, int skip) {
    std::vector<float> l(1), r(1);
    double acc = 0.0;
    for (int i = 0; i < samples; ++i) {
        inst.process(nullptr, nullptr, l.data(), r.data(), 1);
