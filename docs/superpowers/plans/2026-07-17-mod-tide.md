# MOD & TIDE — Modulation greifbar machen: Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Drei klare Regler-Jobs: MOD (ex-DEPTH, pro Deck) skaliert nur noch die vier Textur-Lanes + FX-Targets, RANGE nur noch den Melodie-Ambitus (PITCH-Lane), und ein neuer globaler TIDE-Knob im Center skaliert die Textur-Lane-Rate beider Decks ¼×–4× um eine bit-identische Mitte. Melodie und Rhythmus sind unter MOD und TIDE invariant.

**Architecture:** `Part::target_raw()` nimmt die PITCH-Lane aus der `_depth`-Klammer (FX-Targets bleiben drin); `SuperModulator::set_range()` trifft nur noch `_lanes[LANE_PITCH]` (Textur-Lanes bleiben auf Default-Range 1). TIDE ist ein eigener Faktor `_tide_mult` im `SuperModulator`, der sich in `_apply_rate()` mit dem vorhandenen `_mod_scale` (COUPLE/DRIFT) multipliziert — die Melodie-Clock (`_master_hz`, PITCH-Lane) bleibt unberührt. Die TIDE-Leiter (9 reziprok-symmetrische Verhältnisse, Mitte ×1) lebt neben der RATE-Leiter in `mod/divisions.h`, damit Engine und VCV-Tooltip dieselbe Tabelle lesen. VCV: Enum-Rename DEPTH→MOD im Template (Positions-IDs unverändert), `REV_MOD`-Label → WOBL, TIDE ans Ende von PARAMS (Patch-Kompatibilität wie CHOKE/FILT).

**Tech Stack:** C++17 portable engine (`engine/`), doctest (`tests/`, clang+ninja via `env.sh`), VCV Rack SDK 2.6.6 host (`host/vcv/`, MSYS2 make + WinLibs gcc), panel generated by `host/vcv/res/gen_panel.py`.

```cpp
// TIDE: texture-lane rate scale (spec 2026-07-17 mod-tide). 9 rungs,
// reciprocal-symmetric so the knob centre (index 4) is exactly x1 — that is
// what makes TIDE 0.5 a bit-identical no-op. Names appear verbatim in the
// VCV TIDE tooltip.
inline constexpr float kTideRatios[] =
    { 0.25f, 1.f/3.f, 0.5f, 2.f/3.f, 1.f, 1.5f, 2.f, 3.f, 4.f };
inline constexpr const char* kTideNames[] =
    { "x1/4", "x1/3", "x1/2", "x2/3", "x1", "x3/2", "x2", "x3", "x4" };
inline constexpr int kTideCount = 9;

inline int tide_index(float norm) {
    return static_cast<int>(clampf(norm, 0.f, 1.f) * (kTideCount - 1) + 0.5f);
}

// free (unsynced) curve: 1/4 .. 4, exactly 1 at centre (2^0)
inline float tide_free(float norm) {
    return std::pow(2.f, 4.f * (clampf(norm, 0.f, 1.f) - 0.5f));
}
```

```cpp
void SuperModulator::_apply_rate() {
    _master_hz = _base_hz * _pitch_scale;
    for (int i = 0; i < LANE_COUNT; ++i) {
        const float s = (i == LANE_PITCH) ? _pitch_scale
                                          : _mod_scale * _tide_mult;
        _lanes[i].set_rate_hz(_base_hz * s * kLaneRatio[i]);
    }
}

void SuperModulator::set_tide(float norm) {
    _tide_norm = clampf(norm, 0.f, 1.f);
    _update_tide();
}

void SuperModulator::_update_tide() {
    _tide_mult = _synced ? kTideRatios[tide_index(_tide_norm)]
                         : tide_free(_tide_norm);
    _apply_rate();
}
```

```cpp
float Part::target_raw(int slot) const {
    // Master MOD (ex-DEPTH) shapes the texture only; the PITCH lane is the
    // anchor and keeps its per-slot depth alone (spec 2026-07-17 mod-tide).
    const float d = (slot == LANE_PITCH) ? 1.f : _depth;
    float mod = _active[slot] ? _mod.lane_output(slot) * d * _tdepth[slot] : 0.f;
    float v = _base[slot] + mod;
    return clampf(v, 0.f, 1.f);
}
```
