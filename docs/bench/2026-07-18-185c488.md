# Bench run 2026-07-18 — `185c488`

Measured on a Daisy Seed (STM32H750). 480000000 Hz core clock, block size 96, dcache+icache, `-ffast-math -funroll-loops`. Block budget 960000 cycles.

## Verdict

**2x4 budget — go/no-go.** The full instrument at its worst case (8 voices, COLOR 4-note on both parts, all FX on, high diffusion, echo at max) costs **156 % of the block budget offline**, and **156 % measured inside a real audio callback**. The anchored figure is the one that decides. **Conclusion: the 2x4 architecture does not fit.** The anchored figure is over 100 % of the block budget, so the design has to shed voices or FX.

**Cost per candidate, relative to one real spotymod voice.**

- `modal_voice` — 6.1x one real voice (44x a bare oscillator kernel)
- `string_voice` — 2.0x one real voice (14x a bare oscillator kernel)
- `resonator` — 5.5x one real voice (40x a bare oscillator kernel)
- `formant_osc` — 0.29x one real voice (2.1x a bare oscillator kernel)
- `vosim_osc` — 0.93x one real voice (6.8x a bare oscillator kernel)
- `harmonic_osc` — 0.48x one real voice (3.5x a bare oscillator kernel)
- `grainlet_osc` — 0.67x one real voice (4.9x a bare oscillator kernel)
- `z_osc` — 0.99x one real voice (7.2x a bare oscillator kernel)
- `variable_shape_osc` — 0.19x one real voice (1.4x a bare oscillator kernel)

**SRAM vs SDRAM.** The grain-read proxy (8 scattered interpolated stereo reads per sample, identical window in both regions) costs **5.3x** in SDRAM against SRAM — this is the M5 texture deck's exposure, measured before the sampler exists. The Oliverb pair reads **1.1x**, and the shortened echo-style streaming walk **3.3x**.

*Figures in this section are quoted to whole percentage points and two significant figures for ratios — honest to what this bench can actually resolve (intra-run jitter of roughly 1700 cycles on a 1.5M-cycle workload, and a cross-build layout shift that moved a 29K-cycle workload by about 7%). The tables below retain full measured precision.*
