# Bench run 2026-07-18 — `0e6eeb5`

Measured on a Daisy Seed (STM32H750). 480000000 Hz core clock, block size 96, dcache+icache, `-ffast-math -funroll-loops`. Block budget 960000 cycles.

## Verdict

**2x4 budget — go/no-go.** The full instrument at its worst case (8 voices, COLOR 4-note on both parts, all FX on, high diffusion, echo at max) costs **165.08 % of the block budget offline**, and **164.44 % measured inside a real audio callback**. The anchored figure is the one that decides: under 100 % the 2x4 architecture fits, over it the design has to shed voices or FX.

**Cost per candidate, relative to one real spotymod voice.**

- `modal_voice` — 6.09x one real voice (44.33x a bare oscillator kernel)
- `string_voice` — 1.94x one real voice (14.10x a bare oscillator kernel)
- `resonator` — 5.54x one real voice (40.37x a bare oscillator kernel)
- `formant_osc` — 0.29x one real voice (2.12x a bare oscillator kernel)
- `vosim_osc` — 0.92x one real voice (6.73x a bare oscillator kernel)
- `harmonic_osc` — 0.48x one real voice (3.50x a bare oscillator kernel)
- `grainlet_osc` — 0.66x one real voice (4.81x a bare oscillator kernel)
- `z_osc` — 0.99x one real voice (7.24x a bare oscillator kernel)
- `variable_shape_osc` — 0.19x one real voice (1.35x a bare oscillator kernel)

**SRAM vs SDRAM.** The grain-read proxy (8 scattered interpolated stereo reads per sample, identical window in both regions) costs **5.28x** in SDRAM against SRAM — this is the M5 texture deck's exposure, measured before the sampler exists. The Oliverb pair reads **1.11x**, and the shortened echo-style streaming walk **3.29x**.
